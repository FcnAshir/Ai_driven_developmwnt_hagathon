# Welcome to RAG Chatbot Documentation

This documentation provides information about the AI RAG (Retrieval Augmented Generation) Chatbot project.

## Overview

This project is a RAG chatbot that:
- Extracts content from sitemaps
- Processes and chunks the text
- Stores embeddings in Qdrant vector database
- Uses Cohere for generating embeddings

## Features

- Sitemap parsing and content extraction
- Text chunking for optimal processing
- Vector storage and retrieval
- AI-powered embeddings using Cohere
