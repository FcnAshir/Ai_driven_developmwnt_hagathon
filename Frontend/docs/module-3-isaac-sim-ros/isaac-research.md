### Core Isaac Technologies
- **NVIDIA Omniverse**: Real-time 3D simulation and collaboration platform
- **USD (Universal Scene Description)**: NVIDIA's scene description and 3D graphics interchange framework
- **RTX Rendering**: Real-time ray tracing for photorealistic rendering
- **PhysX**: NVIDIA's physics simulation engine

### Isaac Sim Features
- **Photorealistic Simulation**: RTX rendering for realistic sensor data
- **Synthetic Data Generation**: High-quality training data for AI models
- **Robot Simulation**: Support for various robot types including humanoids
- **Sensor Simulation**: Accurate simulation of cameras, LiDAR, IMU, force/torque sensors
- **USD Scene Composition**: Flexible scene description and composition

### Isaac ROS Packages
- **Isaac ROS Visual SLAM**: Hardware-accelerated visual SLAM
- **Isaac ROS Apriltag**: High-performance fiducial detection
- **Isaac ROS AprilTag**: Fiducial tracking and pose estimation
- **Isaac ROS CenterPose**: 6D object pose estimation
- **Isaac ROS DNN Inference**: GPU-accelerated neural network inference
- **Isaac ROS Image Pipeline**: GPU-accelerated image processing
- **Isaac ROS Stereo Dense Depth**: Depth estimation from stereo cameras
- **Isaac ROS NITROS**: NVIDIA Isaac Transport and ROS SW (data transport optimization)

### Perception Stack
- **Hardware Acceleration**: GPU acceleration for computer vision tasks
- **Deep Learning**: Integration with NVIDIA's AI frameworks
- **Sensor Fusion**: Combining multiple sensor modalities
- **Real-time Performance**: Optimized for real-time robotics applications

### Isaac for Humanoid Robotics
- **Humanoid Models**: Support for complex humanoid robot models
- **Physics Simulation**: Accurate physics for humanoid locomotion
- **Perception Pipeline**: GPU-accelerated perception for humanoid tasks
- **Navigation**: Advanced navigation for humanoid robots
- **Manipulation**: Dextrous manipulation simulation
